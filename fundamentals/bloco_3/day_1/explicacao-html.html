Explicação HTML

O HTML é linguagem de marcação de texto Seu surgimento teve com a comunidade cientifica com objetivo de ajudar no compartilhamento de informação

Criadores

Tim Bernes Lee Robert Cailiau

Versões

  • 2.0 - 1995
  • 3.2 - 1997
  • 4.01 - 1999
  • HTML 5 - 2014

Estrutura primária ou esqueleto de página, possuí tags que representam informações A estrutura de tag é composta por 

    < aberturaComNomeTag atributo="valor">
            conteudo da tag
    < fechamentoTag >
  • DOCTYPE = Navegador interpreta que um arquivo HTML
  • html = raiz da página atributo lang navegador interpreta linguagem
  • head = metadados e configurações da página
  • body = conteudo da página
